Linked List Visualization

How to use:

  1. Enter a value in the input field.
  2. Click the "Add First", "Add Last", or "Add At" buttons to add the value to the linked list.
  3. Click the "Remove First", "Remove Last", or "Remove At" buttons to remove elements from the list.
  4. Click the "Get At" button to retrieve the value at a specific index.
  5. Click the "Contains" button to check if the list contains a specific value.
  6. Click the "Clear" button to remove all elements from the list.

Instruções

Comandos disponíveis:

  • ADICIONAR_INICIO [valor] - Adiciona o valor ao início da lista
  • ADICIONAR_FIM [valor] - Adiciona o valor ao fim da lista
  • REMOVER_INICIO - Remove o primeiro valor
  • REMOVER_FIM - Remove o último valor
  • OBTER [índice] - Exibe o valor no índice especificado
  • CONTER [valor] - Verifica se o valor está na lista
  • LIMPAR - Limpa todos os valores da lista
